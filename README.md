# Honey Bee Idle Preview

This repository contains the public static preview build for Honey Bee Idle.

## Local run

```bash
python3 -m http.server 8000
```

Then open:

```text
http://127.0.0.1:8000/
```

## GitHub Pages

This repository is intended to deploy with GitHub Pages from GitHub Actions.
Enable Pages with `Source = GitHub Actions`, then push to `main`.
